# Source Code

This folder contains the Python implementation of the hybrid machine learning framework for electricity theft detection.

## Files

- `preprocessing.py` — Data loading, cleaning, and train/test preparation.
- `feature_engineering.py` — Feature preparation and integration of load and frequency features.
- `train_model.py` — Training of machine learning models and hybrid ensemble approach.
- `evaluate.py` — Model performance evaluation and visualisation.
- `utils.py` — Utility functions used across the project.
