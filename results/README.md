This folder contains the experimental results generated from the hybrid machine learning framework for electricity theft detection in smart grids.

The results include model performance evaluation, comparison metrics, and visual analysis outputs.

## Contents

### Model Performance

- `model_comparison.csv`
Contains numerical performance metrics for the evaluated machine learning models.

- `model_comparison_table.png`
Visual representation of the model comparison results.

### Performance Visualisations

- `bar_comparison.png`
Comparison of model performance metrics.

- `recall_comparison.png`
Recall comparison between evaluated models.

- `roc_curve.png`
ROC curve analysis showing classifier discrimination performance.

### Error Analysis

- `confusion_matrix_hybrid.png`
Confusion matrix of the proposed hybrid model showing classification performance between theft and non-theft cases.

## Evaluation Metrics

The framework evaluates performance using:

- Accuracy
- Recall
- F1-score
- ROC-AUC

All generated results are obtained from the machine learning evaluation pipeline implemented in the `src` directory.
